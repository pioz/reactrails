## Installation

Add this line to your application's Gemfile:

```ruby
gem "reactrails"
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install reactrails
```

## Dev

Bundle the react_server_rendering.js with the command `yarn build`
